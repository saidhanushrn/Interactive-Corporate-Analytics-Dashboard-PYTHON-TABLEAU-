# Interactive-Corporate-Analytics-Dashboard-PYTHON-TABLEAU
Scraped and processed datasets covering 100 top companies including revenue, employee numbers, and headquarters locations. 
Developed an interactive Tableau dashboard featuring 10+ industry sectors with combined filters, dynamic actions, and geographic maps. 
Implemented advanced Tableau techniques: combined filters, dynamic dashboard actions, geographic mapping, and custom tooltips for deep user-driven exploration. 
Delivered insights on industry dominance, company growth, and efficiency, enabling executive-level analytics and data storytelling. 
Demonstrated end-to-end workflow from web scraping to visualization, improving data accessibility, and storytelling for decision-makers.
