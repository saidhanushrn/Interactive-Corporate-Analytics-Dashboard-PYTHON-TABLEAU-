# Interactive-Corporate-Analytics-Dashboard-PYTHON-TABLEAU
●	Scraped, cleaned, and transformed company and revenue datasets from Wikipedia using Python and Pandas.
●	Developed an interactive Tableau dashboard showcasing top industry leaders, company revenue, employee counts, and headquarters locations.
●	Implemented advanced Tableau techniques: combined filters, dynamic dashboard actions, geographic mapping, and custom tooltips for deep user-driven exploration.
●	Delivered insights on industry dominance, company growth, and efficiency, enabling executive-level analytics and data storytelling.
●	Demonstrated full-cycle data workflow: web scraping, data wrangling, visualization, and business insight communication.
